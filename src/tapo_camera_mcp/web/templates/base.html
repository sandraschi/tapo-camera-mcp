Platform Overview

The Home Security MCP unifies camera control, sensor telemetry, and observability in a single orchestration plane.

  • Use Dashboard for real-time situational awareness.
  • Edge agents forward metrics and logs for every node.
  • Grafana dashboards and Alertmanager fast-track incident triage.

Quick Start

  • üé• Add camera via MCP tool add_camera.
  • üîå Configure Tapo P115 credentials in config.yaml.
  • üìä Import Grafana provisioning bundle for dashboards.

Camera Operations

  • Start stream: Dashboard ‚Üí Start Stream or API /api/cameras/{id}/stream.
  • Snapshots: /api/cameras/{id}/snapshot (returns JPEG).
  • PTZ capabilities exposed via MCP tools move_ptz, save_ptz_preset.
  • Status metrics: Prometheus series tapo_camera_status, tapo_camera_stream_bitrate.

Troubleshooting

  • Verify credentials in config.yaml and secrets backend.
  • Check live logs for stream pipeline entries.
  • Run chaos scenario CT-02 to validate auto-recovery.

Sensor & Energy Telemetry

  • Tapo P115 ingestion via tapo_p115 adapter using python-kasa.
  • REST endpoints: /api/sensors/tapo-p115, history with /history?hours=.
  • Grafana dashboard: Sensor Overview (power, costs, energy).
  • Add Netatmo / Nest Protect ingestion through edge collectors.

Data Hygiene

  • Validate metrics in Prometheus: tapo_p115_power_watts, tapo_p115_daily_cost.
  • Rotate secrets using plan in SECRETS_HARDENING_PLAN.md.

Observability Stack

  • Prometheus federation config: deploy/prometheus/prometheus.yaml.
  • Loki & Promtail shipping structured logs with semantic labels.
  • CI validates configs via observability workflow job.
  • Dashboards: Camera Overview, Sensor Overview, Logging Overview.
  • Alerts: CameraOffline, EdgeAgentMissedScrape routed through Alertmanager.

Refer to RUNBOOKS.md for restart procedures, scaling guidance, and escalation paths.

  • Pluggable recipe library for device onboarding and energy automation.
  • Future roadmap: autop-run scene detection, ai-vision alerts, and policy aware automation.
  • Document templates available under docs/mcp-technical/.
