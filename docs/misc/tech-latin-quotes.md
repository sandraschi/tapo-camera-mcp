# Tech Latin Quotes & Proverbs

A collection of programming wisdom, jokes, and proverbs expressed in Latin.

## Programming Wisdom

- **"Primum compilare, deinde philosophari"**
  - "First compile, then philosophize"
  - A play on 'primum vivere, deinde philosophari' (first live, then philosophize)

- **"Dum compilo, spero"**
  - "While I compile, I hope"
  - Based on 'dum spiro, spero' (while I breathe, I hope)

- **"O tempora, o mores, o documenta!"**
  - "Oh the times, oh the customs, oh the documentation!"
  - Cicero's famous complaint, repurposed for poor documentation

## Programming Jokes

- **"Veni, Vidi, Debuggi"**
  - "I came, I saw, I debugged"
  - A play on Caesar's 'Veni, vidi, vici'

- **"Cogito, ergo bug"**
  - "I think, therefore I have bugs"
  - A play on Descartes' 'Cogito, ergo sum'

- **"Alea i acta est"**
  - "The i has been incremented"
  - A play on Caesar's 'the die has been cast', but for loop iterations

## Error Messages

- **"Codus Mortuus Est"**
  - "The code is dead"
  - When the program crashes completely

- **"Memoria Fugit, Tempus Deficit"**
  - "Memory escapes, time runs out"
  - Out of memory error with timeout

- **"Nullus Nullor Nullissimus"**
  - "None, Noner, Nonest"
  - When you get null pointer exceptions everywhere

## Tech Proverbs

- **"Backup semper fidelis"**
  - "Backups are always faithful"
  - Based on the Marine Corps motto 'Semper Fidelis'

- **"Git committere humanum est, git revertere divinum"**
  - "To git commit is human, to git revert divine"
  - Based on 'Errare humanum est' (to err is human)

- **"Carpe Datum"**
  - "Seize the data"
  - A play on 'Carpe Diem' (seize the day)

## Documentation Wisdom

- **"Commentarium Scriptum, Problema Solutum"**
  - "Write the comment, solve the problem"
  - The importance of documentation

- **"Lorem Ipsum Deletum Est"**
  - "The placeholder text has been deleted"
  - When finally replacing placeholder text

- **"Documenta Et Labora"**
  - "Document and work"
  - Based on the Benedictine motto 'Ora et Labora' (pray and work)

## Debugging Wisdom

- **"Testus Unus, Testus Nullus"**
  - "One test is no test"
  - The importance of thorough testing

- **"In Absentia Lucis, Tenebrae Vincunt"**
  - "In the absence of light mode, dark mode prevails"
  - The eternal IDE theme debate

- **"Deus Ex Machina Learning"**
  - "God from the Machine Learning"
  - When ML magically solves your problem

## Deployment Prayers

- **"Pater Noster, Qui Es In Servus"**
  - "Our Father, Who Art In Server"
  - A system administrator's prayer

- **"Deploy In Pace"**
  - "Deploy in peace"
  - A peaceful deployment wish

- **"Status 200 Vobiscum"**
  - "Status 200 be with you"
  - A blessing for web developers

## Philosophical Tech Quotes

- **"Quidquid Latine Dictum Sit, Altum Videtur"**
  - "Whatever is said in Latin sounds profound"
  - A meta-joke about this entire collection

- **"Si Hoc Legere Scis, Nimium Programmaverunt"**
  - "If you can read this, you've programmed too much"
  - For those who see code everywhere

- **"Semper Tyrannis Typographicis"**
  - "Always with typographical tyranny"
  - The eternal struggle with formatting
