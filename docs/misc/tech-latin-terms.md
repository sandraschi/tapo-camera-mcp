# Tech Latin Terms Dictionary

A comprehensive dictionary of technology terms translated into Latin, complete with pronunciations and definitions.

## Intellectus Artificialis (Artificial Intelligence)

### Neural Networks & Deep Learning
- **artificial intelligence**: intellectus artificialis (in-tel-LEK-tus ar-ti-fi-ki-AH-lis)
  - The science and engineering of making intelligent machines

- **neural network**: rete neuronium simulatum (RE-te new-RO-ni-um si-mu-LAH-tum)
  - A computing system inspired by biological neural networks

- **deep learning**: disciplina profunda (dis-ki-PLI-na pro-FUN-da)
  - Machine learning based on deep neural networks

- **language model**: modellum linguae (mo-DEL-lum LIN-gwae)
  - AI system designed to understand and generate human language

- **transformer**: transformator attentionis (trans-for-MA-tor at-ten-ti-O-nis)
  - Neural network architecture using attention mechanisms

- **prompt**: suggestio directiva (sug-GES-ti-o di-rek-TI-va)
  - Input text that guides AI behavior

- **fine-tuning**: calibratio subtilis (ka-li-BRA-ti-o SUB-ti-lis)
  - Process of adjusting a pre-trained model for specific tasks

- **hallucination**: confabulatio artificialis (kon-fa-bu-LA-ti-o ar-ti-fi-ki-A-lis)
  - When AI generates false or nonsensical information

- **embedding**: immersio vectorialis (im-MER-si-o vek-to-ri-A-lis)
  - Mathematical representation of data in vector space

- **tokenization**: segmentatio vocabularis (seg-men-TA-ti-o vo-ka-bu-LA-ris)
  - Process of breaking text into smaller units

## Fundamentalia Computatri (Computer Basics)

### Core Computing Concepts
- **computer**: computatrum (kom-poo-TAH-trum)
  - The basic computing machine

- **binary**: binarius digitalis (bi-NA-ri-us di-gi-TA-lis)
  - Digital binary, the fundamental language of computers

- **bit**: minimum digitalis (MI-ni-mum di-gi-TA-lis)
  - Digital minimum, the smallest unit of data

- **byte**: octetus digitalis (ok-TE-tus di-gi-TA-lis)
  - Digital octet, a group of eight bits

## Apparatus Principalis (Main Hardware)

### Core Computer Components
- **laptop/notebook computer**: computatrum portabilis (kom-poo-TAH-trum por-TAH-bi-lis)
  - A portable computing device, literally "portable computer"

- **hard disk drive**: discus duratus memoriae (DIS-kus du-RAH-tus me-MO-ri-ae)
  - Hard disk for memory storage, literally "hardened disk of memory"

- **RAM**: memoria volatilis instantanea (me-MO-ri-a vo-LA-ti-lis in-stan-TA-ne-a)
  - Volatile instant memory, emphasizing its temporary nature

- **GPU**: processorum graphicum (pro-ces-SO-rum GRA-fi-kum)
  - Graphics processor, literally "graphical processor"

- **motherboard**: tabula mater principalis (TA-bu-la MA-ter prin-ki-PA-lis)
  - Main motherboard, literally "principal mother board"

- **CPU**: cerebrum computatrum (KE-re-brum kom-poo-TAH-trum)
  - Computer brain, emphasizing its central role in processing

- **SSD**: discus solidus velocis (DIS-kus SO-li-dus ve-LO-kis)
  - Fast solid disk, highlighting its speed and solid-state nature

- **cooling fan**: ventilator frigidarius (ven-ti-LA-tor fri-gi-DA-ri-us)
  - Cooling fan, literally "cooling ventilator"

- **power supply**: fons potentiae (fons po-TEN-ti-ae)
  - Power source, literally "fountain of power"

### Peripherals & Input Devices
- **keyboard**: tabulatum digitalis (ta-bu-LA-tum di-gi-TA-lis)
  - Digital keyboard, literally "digital tablet"

- **mouse**: mus digitalis (mus di-gi-TA-lis)
  - Digital mouse, maintaining the familiar animal metaphor

- **monitor/display**: speculum electronicum (SPE-ku-lum e-lek-TRO-ni-kum)
  - Electronic mirror/display, emphasizing its visual output nature

### Peripheral Devices
- **printer**: impressorum digitalis (im-pre-SO-rum di-gi-TA-lis)
  - Digital printer, for creating physical copies

- **scanner**: ocularus digitalis (o-ku-LA-rus di-gi-TA-lis)
  - Digital eye, for converting physical to digital

- **webcam**: camera retialis (KA-me-ra re-ti-A-lis)
  - Network camera, for web communication

## Programma (Software)

### Core Software Concepts
- **operating system**: systema operandi (SIS-te-ma o-pe-RAN-di)
  - System of operation, the fundamental software layer

- **program/application**: programma applicatum (pro-GRAM-ma ap-pli-KA-tum)
  - Applied program, software for specific tasks

- **algorithm**: algorithmus logicus (al-go-RIT-mus LO-gi-kus)
  - Logical procedure for solving problems

- **database**: bibliotheca digitalis (bi-bli-o-TE-ka di-gi-TA-lis)
  - Digital library, for structured data storage

- **compiler**: translator machinae (trans-LA-tor ma-KI-nae)
  - Machine translator, converts high-level code to machine code

- **debugger**: purgator errorum (pur-GA-tor er-RO-rum)
  - Error purger, for finding and fixing code problems

- **user interface**: interfacies utentis (in-ter-FA-ki-es u-TEN-tis)
  - User interface, the point of human-computer interaction

## Errores (Errors & Warnings)

### Common Error Messages
- **404 error**: Pagina Non Inventa (pa-GI-na non in-VEN-ta)
  - Page not found

- **500 error**: Servitor Confusus Est (ser-VI-tor kon-FU-sus est)
  - Server is confused

- **undefined**: Res Indefinita (res in-de-fi-NI-ta)
  - Undefined thing

- **null**: Nihilum Digitale (NI-hi-lum di-gi-TA-le)
  - Digital nothing

- **memory leak**: Memoria Sicut Aqua Effluit (me-MO-ri-a SI-kut A-kwa EF-flu-it)
  - Memory flows away like water

## Retialis (Networking)

### Network Fundamentals
- **internet**: rete mundialis (re-TE mun-di-A-lis)
  - World network, the global computer network

- **router**: director retialis (di-REK-tor re-ti-A-lis)
  - Network director, guides data through networks

- **firewall**: murus protectivus (MU-rus pro-tek-TI-vus)
  - Protective wall, for network security

- **server**: servitor digitalis (SER-vi-tor di-gi-TA-lis)
  - Digital servant, provides services to other computers

- **packet**: fasciculus datorum (fas-KI-ku-lus da-TO-rum)
  - Data bundle, a unit of network transmission

- **protocol**: protocollum communicationis (pro-to-KOL-lum ko-mu-ni-ka-ti-O-nis)
  - Communication protocol, rules for data exchange
